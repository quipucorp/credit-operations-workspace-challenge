# Centro de Operaciones de Crédito

Repositorio starter para un take-home challenge de Semi Senior Full Stack usando:

- Angular + Ionic + TypeScript
- Node.js + TypeScript

## Qué Incluye

- `frontend/`: aplicación Angular + Ionic
- `backend/`: API simple en Node.js + TypeScript
- `docs/challenge.md`: consigna para el challenge
- `docs/references/`: referencias visuales para tareas opcionales de UI

## Cómo Correr el Proyecto

### Backend

```bash
cd backend
npm install
npm run dev
```

URL local del backend:

```bash
http://localhost:4000
```

### Frontend

```bash
cd frontend
npm install
npm start
```

URL local del frontend:

```bash
http://localhost:4200
```
